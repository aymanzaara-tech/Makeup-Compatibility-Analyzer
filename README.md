## Makeup Compatibility Analyzer

## Overview

Makeup Compatibility Analyzer is a web application designed to help users evaluate whether their selected makeup products work well together. The application analyzes factors such as skin type, skin tone, undertones, and occasion to provide compatibility scores, recommendations, and warnings.

## Problem Statement

Many makeup users purchase products individually without knowing whether they complement each other as a complete look. This often leads to mismatched products, wasted purchases, and trial-and-error experimentation.

The goal of this project is to provide users with a data-driven compatibility analysis that helps them make informed makeup decisions.

## Key Features

- User Registration and Login
- Personalized User Profiles
- Makeup Product Selection
- Compatibility Analysis Engine
- Compatibility Scoring System
- Personalized Recommendations
- Product Harmony Warnings
- Saved Analysis History
- Responsive User Interface
